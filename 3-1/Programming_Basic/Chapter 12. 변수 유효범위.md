## exturn 키워드

- 블록이나 함수 내부에서 이미 전역 변수로 선언된 변수를 사용하고자 할 때 사용

- 선언이 더 위에 있으면 생략 가능

## static 키워드

- 정적 변수 선언 시 사용

- 정적 전역변수는 선언한 파일에서만 사용 가능

## register 키워드

- 저장공간을 레지스터로 할당
- 지역변수와 동일

## 메모리의 세 영역

- 스택 : 지역 변수
- 데이터 : 전역 변수, 정적 변수
- 힙 : 동적 할당 변수
